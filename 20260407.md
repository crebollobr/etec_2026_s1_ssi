
# 🧪 Laboratório: Teste de Segurança com OWASP ZAP

## 📁 1. Preparação do ambiente

Abra a pasta **Downloads** e organize tudo nela.

---

## 📥 Downloads necessários

### 🔹 OWASP ZAP (ZIP)

👉 [https://www.zaproxy.org/download/](https://www.zaproxy.org/download/)
Baixar: **Cross Platform Package**

* Arquivo: `ZAP_2.17.0_Crossplatform.zip`
* Extrair em **Downloads**
* Durante a extração, **apagar o nome da pasta**
* Renomear a pasta final para:

```text
zap
```

---

### 🔹 Java (ZIP portátil)

👉 [https://adoptium.net/pt-BR/temurin/releases](https://adoptium.net/pt-BR/temurin/releases)

Baixar:

* **Windows**
* **Temurin 25 (LTS)**
* Arquivo ZIP

Exemplo:

```text
OpenJDK25U-jdk_x64_windows_hotspot_25.0.2_10.zip
```

* Extrair em **Downloads**
* Durante a extração, **apagar o nome da pasta**
* Renomear para:

```text
java
```

---

### 🔹 Aplicação vulnerável

Criar pasta:

```text
Downloads\web
```

Baixar:
👉 [https://github.com/crebollobr/etec_2026_s1_ssi/blob/main/app.py](https://github.com/crebollobr/etec_2026_s1_ssi/blob/main/app.py)

* Clique em **Raw**
* Salvar como:

```text
web\app.py
```

---

## 📁 Estrutura final

```text
Downloads\
 ├── web\
 │    └── app.py
 ├── java\
 │    └── bin\java.exe
 └── zap\
      └── zap-2.17.0.jar
```

# 🌐 4. Teste do site

1. Abrir o navegador
2. Acessar:

```
http://localhost:5000
```

3. Testar login com qualquer valor

---

# 🕷️ 5. Varredura com ZAP

No ZAP:

1. Aba **Início Rápido**
2. Clique em **Varredura Automatizada**
3. Em URL:

```
http://localhost:5000
```

4. Clique em **Ataque**

---

# ⚔️ 6. Exploração

1. Aba **Sites**
2. Clique com botão direito em:

```
http://localhost:5000
```

3. Vá em:

```
Ataque → Varredura Ativa
```

4. Aguarde terminar

---

# 🚨 7. Resultado esperado

Ir na aba **Alertas**

✔ Deve aparecer vulnerabilidade:

```
SQL Injection
```

---

# 🧠 8. Análise

Clique no alerta e observe:

* Requisição enviada
* Payload usado

Exemplo:

```sql
admin' OR '1'='1' --
```

# ⚠️ Regras importantes

* Tudo deve estar dentro de `Downloads`
* A pasta deve se chamar `java`
* Deve existir: `java\bin\java.exe`
* Não pode ter pasta extra (ex: `jdk-xx`)
* Não usar `%%` no caminho
* Usar seu login: `login_aluno`
